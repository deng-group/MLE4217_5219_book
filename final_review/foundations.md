# Foundations: Programming, Git, and Computer and Computation

## What we covered

- Core Python ideas used throughout the course, including variables, functions, NumPy, plotting, and reading short code snippets.
- Practical version-control habits with git and GitHub for classwork and collaboration.
- Basic computing concepts including hardware, software, memory hierarchy, CPU/GPU roles, and HPC ideas.
- The broader role of computation and data in materials informatics.

## What you should be able to do

- Read a short Python or NumPy snippet and explain the result in words.
- Recognize what makes a plot clear and scientifically readable.
- Explain the meaning of commit, branch, merge, conflict, pull request, and remote synchronization.
- Describe the practical difference between CPU, GPU, memory, storage, and HPC resources.
- Explain how computing tools support later modelling and data-analysis tasks in the course.
